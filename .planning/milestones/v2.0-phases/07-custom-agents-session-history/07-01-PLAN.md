   - Helper `ensureUniqueSlug(slug: string)`: query prisma for existing, append -2/-3 etc. on conflict

   - `PATCH` handler:

     - Call `agentService.delete(id)`
     - Catch "has sessions" error, return 409 with error message
     - Catch "not custom" error, return 403
     - Return 200 with `{ success: true }`
