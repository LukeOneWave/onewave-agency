  **Step 1: Add getPendingReview to dashboardService** (src/lib/services/dashboard.ts)

  Add a new method to the existing dashboardService object:

  ```typescript
  async getPendingReview(limit: number = 5) {
    return prisma.deliverable.findMany({
      where: { status: "pending" },
      take: limit,
      orderBy: { createdAt: "desc" },
      include: {
        message: {
          select: {
            content: true,
            session: {
              select: {
                id: true,
                title: true,
                agent: {
                  select: { name: true, slug: true, color: true },
                },
              },
            },
          },
        },
      },
    });
  }
  ```

  Use a single findMany with nested include to avoid N+1 queries (Pitfall 2 from research). The return type is inferred by Prisma -- do NOT manually define it. Export a type alias if needed:
  `export type PendingDeliverable = Awaited<ReturnType<typeof dashboardService.getPendingReview>>[number];`
