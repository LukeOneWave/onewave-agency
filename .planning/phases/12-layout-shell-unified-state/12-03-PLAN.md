2. Add a new `useEffect` after the existing localStorage hydration effect (after line 60). This effect watches `panelOpen` and calls the imperative handle:

```typescript
// Sync store panelOpen state to the visual panel via imperative ref
useEffect(() => {
  const panel = artifactsPanelRef.current;
  if (!panel) return;
  if (panelOpen && panel.isCollapsed()) {
    panel.expand();
  } else if (!panelOpen && !panel.isCollapsed()) {
    panel.collapse();
  }
}, [panelOpen]);
```

The guard `panel.isCollapsed()` prevents calling collapse when already collapsed (which is a no-op per the library docs but avoids unnecessary calls) and prevents infinite loops since `onResize` syncs state back to the store when the panel collapses/expands via drag.

Do NOT change the `]` key handler -- it should continue to call `useChatStore.getState().togglePanel()` which flips `panelOpen`, and the new `useEffect` reacts to that change.
