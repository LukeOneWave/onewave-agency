1. **Create src/app/api/chat/session/route.ts**: POST endpoint to create a new chat session:
   - Accepts `{ agentId: string, model?: string }`
   - Calls `chatService.createSession(agentId, model || "claude-sonnet-4-6")`
   - Returns JSON `{ sessionId: session.id }`
   - This is needed so the client can create a session and then navigate to `/chat/[sessionId]`

2. **Create src/app/chat/[sessionId]/page.tsx**: Server component page:
   - Params: `{ sessionId: string }`
   - Loads session via `chatService.getSession(sessionId)` — if not found, call `notFound()`
   - Renders `<ChatPage session={session} />` passing the full session with agent and messages
   - Set metadata title to `"Chat with {agent.name} | OneWave"`

3. **Create src/components/chat/ChatPage.tsx**: Client component ("use client"):
   - Props: `session` (ChatSession with agent and messages included)
   - On mount (useEffect): calls `useChatStore.initSession(session.id, session.agent.slug, session.agent.name, session.messages)`
   - Layout: Full height flex column. Top bar with agent name, division badge, and `<ModelSelector />`. Main area with `<MessageList />`. Bottom with `<ChatInput />`.
   - Show `<StreamingIndicator />` when `isStreaming` is true
   - Show error toast via sonner when `error` changes (useEffect)

4. **Create src/components/chat/MessageList.tsx**: Client component:
   - Reads `messages` from `useChatStore`
   - Renders each message as `<MessageBubble />`
   - Auto-scrolls to bottom on new messages (useEffect with ref on container, scrollIntoView on messages length change)
   - Shows empty state when no messages: "Send a message to start the conversation"

5. **Create src/components/chat/MessageBubble.tsx**: Client component:
   - Props: `role: string`, `content: string`
   - User messages: right-aligned, `bg-primary text-primary-foreground` rounded bubble, plain text
   - Assistant messages: left-aligned, `prose prose-invert max-w-none` div with:
     ```tsx
     import ReactMarkdown from "react-markdown";
     import remarkGfm from "remark-gfm";
     import rehypeHighlight from "rehype-highlight";
     import "highlight.js/styles/github-dark.css";
     ```
     Render content with `<ReactMarkdown remarkPlugins={[remarkGfm]} rehypePlugins={[rehypeHighlight]}>{content}</ReactMarkdown>`
   - Handle empty content (streaming placeholder): show `<StreamingIndicator />` inline

6. **Create src/components/chat/ChatInput.tsx**: Client component:
   - Textarea (not input — for multiline) with send button (lucide Send icon)
   - Submit on Enter (without Shift), Shift+Enter for newline
   - Calls `useChatStore.sendMessage(content)` on submit
   - Disabled when `isStreaming` is true (both textarea and button)
   - Show stop button (lucide Square icon) when streaming, calls `stopStreaming()`
   - Auto-focus on mount

7. **Create src/components/chat/ModelSelector.tsx**: Client component:
   - Import `CLAUDE_MODELS` from `@/types/chat`
   - Use shadcn `Select` component (or a simple dropdown with `<select>` if shadcn Select not available)
   - Reads `selectedModel` from `useChatStore`, calls `setModel` on change
   - Disabled when `isStreaming` is true
   - Show model name and description in options

8. **Create src/components/chat/StreamingIndicator.tsx**: Simple animated dots component:
   - Three dots with CSS animation (opacity pulse)
   - Tailwind-only animation, no external library
